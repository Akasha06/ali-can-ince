## 2) Update links
Open `index.html` and replace these:
- LinkedIn URL
- GitHub URL (optional)
- Email address

Search for `href="#"` and update.

## 3) Publish (quick)
### GitHub Pages
1. Create a repository (e.g., `can-portfolio`)
2. Upload all files
3. Settings → Pages → Deploy from main branch

### Cloudflare Pages
1. Create a new Pages project
2. Upload folder
3. Deploy
